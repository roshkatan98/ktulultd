# Legal

## Scope

This repository contains a mirrored static copy of a publicly accessible website as requested by the site operator.

## Data Handling

- No user database is stored in this project
- No backend personal-data processing is implemented here

## Privacy And Compliance

- Israeli privacy law considerations apply if future forms or analytics are added
- GDPR and other international privacy requirements apply if EU visitors are tracked or personal data is collected
- User consent should be obtained before adding non-essential tracking

## API Terms

- Current mirrored page still depends on third-party Wix-hosted runtime assets
- Continued use must comply with the relevant third-party service terms

## Data Retention

- This project currently stores only static website assets

## Security Principles

- No secrets should be committed
- Deployment should remain read-only static hosting where possible
